## AYUDA

Si un enlace crea una nota nueva en lugar de abrir la existente, el nombre del archivo no coincide con el texto del enlace.  
Solución: usar alias para vincular correctamente.

Ejemplo:
[[moc-devops|MOC - DevOps]]

Esto muestra “MOC - DevOps” pero apunta al archivo `moc-devops.md`.  
Convención: nombres de archivo en minúsculas, sin espacios; alias para formato visible.
